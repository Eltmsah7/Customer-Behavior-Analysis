# Customer Shopping Behavior Analysis 🛒📊

## 📌 Project Overview
This project provides a comprehensive end-to-end data analysis to understand consumer shopping patterns. The objective is to identify key drivers of customer decisions—such as discounts, reviews, and payment preferences—to help stakeholders optimize marketing strategies and improve customer retention.

## 🛠️ Tech Stack
- **Python (Pandas, NumPy, Seaborn):** For data cleaning, handling missing values, and feature engineering (Age grouping, frequency mapping).
- **SQL (T-SQL):** For advanced querying, customer segmentation, and performance metrics.
- **Power BI:** For building an interactive dashboard and visualizing business insights.
- **SQLAlchemy:** To automate the data pipeline from Python to SQL Server.

## 📈 Business Problems Solved
The analysis answers critical business questions including:
- **Revenue Impact:** Comparison of revenue generated by gender and age groups.
- **Customer Segmentation:** Classifying users into 'New', 'Returning', and 'Loyal' based on purchase history.
- **Shipping Influence:** Analyzing if faster shipping methods correlate with higher order values.
- **Subscription Analysis:** Evaluating the spend behavior of subscribed vs. non-subscribed members.

## 📂 Project Structure
- `customer_shipping_behavior.ipynb`: Python notebook for EDA and Data Preprocessing.
- `Customer_Behaviors_analysis.sql`: SQL script containing 10+ business-critical queries.
- `Business_Problem_Document.pdf`: Detailed project requirements and objectives.
- `Dashboard.pbix`: (Optional) Power BI file for interactive visualization.

## 💡 Key Insights
- **High-Value Discount Users:** Identified a segment of customers who utilize discounts but consistently spend above the average purchase amount.
- **Loyalty Trends:** Customers with more than 5 previous purchases are significantly more likely to subscribe to premium services.
- **Category Leaders:** Determined the top 3 products in each category based on order volume using SQL Window Functions.

## 🚀 How to Run
1. Clone the repository.
2. Run the Python notebook to clean the `customer_shopping_behavior.csv`.
3. Use the SQLAlchemy engine in the script to upload data to your SQL Server.
4. Execute the SQL queries provided in the `.sql` file for deep-dive analysis.
